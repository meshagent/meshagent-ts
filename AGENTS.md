# UI Library Agent Guide

Start here when choosing packages:

## Primary packages
- @meshagent/meshagent: core api client to meshagent
- @meshagent/meshagent-auth: authenticate to meshagent
- @meshagent/meshagent-react: react UI hooks for connecting to room and meshagent document
- @meshagent/meshagent-react-dev: react UI implements dev console
- @meshagent/meshagent-tailwind: react UI implements chat, file preview, forms, meetings components with tailwind & shadcn UI library
